# 交互设计项目

这是一个包含多个页面的交互设计项目，包括以下功能：

## 页面列表

1. **发现.html** - 产品首页，提供信息陈列和检索功能
   - 搜索功能（支持标题、标签、内容搜索，高亮显示）
   - 资讯列表展示
   - 热门文章TOP20

2. **早报.html** - 早报列表页
   - 展示本期早报和往期早报
   - 支持订阅配置
   - 分享早报标识

3. **早报详情.html** - 早报详情页
   - AI智能总结
   - TOP10热门事件列表
   - 支持编辑和取消订阅

4. **订阅配置.html** - 早报订阅配置页
   - 事件类型选择
   - 行业标签和实体标签选择
   - 推送规则配置（周期、时间、分享人）

5. **详情页.html** - 资讯详情页
   - 文章标题、媒体、发布时间
   - 行业标签、实体标签
   - 文章摘要和正文
   - 查看原文按钮

6. **管理员配置.html** - 管理员配置页
   - 事件分类管理（事件类型、行业、实体）
   - 数据源管理

## 功能特点

- 现代化UI设计
- 响应式布局
- 完整的页面导航
- 数据持久化（localStorage）
- 搜索和筛选功能
- AI智能总结

## 使用说明

1. 打开 `发现.html` 作为首页入口
2. 通过底部导航在不同页面间切换
3. 可在设置页面配置早报订阅规则

## 技术栈

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage
